## Paiza Cランク
`values` ハッシュのキーだけを取得するメソッド

```
h = {a: 1, b: 2, c: 3}
p h.values   # => [1, 2, 3]

```

## React
スプレッド構文（JS）配列式や文字列などの反復可能オブジェクトを展開することができる。
https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Operators/Spread_syntax

```
const arr = [1, 2, 3];
const newArr = [...arr, 4, 5];
console.log(newArr); // [1, 2, 3, 4, 5]

```

Rubyでいうと、最近だと、paizaでeach文で回して配列に追加してたりするものをスプレッド構文で代用できるイメージ。

★定数は大文字、連結する場合は_を使う。
