subメソッド 指定したパターンにマッチした最初の部分を特定の文字列に置換。
```
a = "yamada-yamada"
a.sub(/ya/, "ta")
#出力 tamada-yamada

```

gsubメソッド マッチしたすべての部分を特定の文字列に置換
```
a = "yamada-yamada"
a.gsub(/ya/, "ta")
#出力 tamada-tamada
```

複数置換したい場合
```
str = "bagabg"

rules = { "a" => "1", "b" => "3", "g" => "5" }
str = str.gsub(/a|b|g/, rules)
#出力 => "531315"

```
