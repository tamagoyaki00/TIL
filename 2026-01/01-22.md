## オブジェクトクラス

すべてのクラスはObjectクラスを継承

※記述はしてなくても自動的に継承されている

Objectクラスの重要なメソッド

**①boolean equals(Object obj)**

自オブジェクトとobjが等しいかと比べる

例：stu1.equals(stu2)

```ruby
Student stu1 = new Student("田中")
Student stu2 = new Student("田中")

// falseになる

```

ここでは”田中”といオブジェクトを見るのではなく、参照値を比較しているのでfalseになる

**②String to String()**

オブジェクトの情報を文字列で返す。System.out.println(obj)をしたときは自動的にto Stringメソッドが呼び出される

```ruby
class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // toStringメソッドをオーバーライド
    @Override
    public String toString() {
        return "name:" + name + "age:" + age + "}";
    }
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person("Taro", 20);

        // ここで toString() が自動的に呼ばれる
        System.out.println(p);

        // 明示的に呼んでも同じ
        System.out.println(p.toString());
    }
}

// 出力
// name:Taro
// age:20
```

## 文字列の比較に==を使ってはいけない理由

Stringクラス

- “”で囲まれた文字列はStringオブジェクトになる
- ～を使うと同じ文字列のオブジェクトが再利用される

例：自文字列とobjの文字列内容を比べる

```ruby
String s1 = "Java";
String s2 = new String ("Java");
String s3 = "Java";

s1.equals(s2) // true
s1 == s2 // false
s1.equals(s3) //true
s1 == s3 //true
```

## 配列とArrayList

https://docs.oracle.com/javase/jp/25/docs/api/java.base/java/util/ArrayList.html

ArrayList<E>

複数のオブジェクトをまとめて管理できるコレクションの一つ

サイズが動的に変わる配列

格納する要素のデータ型を<>内に指定

【参考メソッド】

boolean add (E obj) : 最後に追加

E get（int i）: i番目の要素を取得

int size(): リストの要素数を取得

Iterator<E> iterator(): イテレータを取得　※以下で説明

```ruby
package chapter6;
import java.util.ArrayList; // インポートをする

public class ArrayListSample {

	public static void main(String[] args) {
		ArrayList<String> list = new ArrayList<String>();
		list.add("Java入門");
		list.add("勉強中");
		list.add("頑張ろう");
		
		for(int i = 0; i < list.size(); i++) {
			System.out.println(list.get(i));
		}

	}

}
```

### Iterator<E>

https://docs.oracle.com/javase/jp/25/docs/api/java.base/java/util/Iterator.html

コレクション内の要素を指し示すカーソルのようなものコレクションクラスのiterator()メソッドで取得できる

【参考メソッド】

hasNext()　次に要素があればtrue

next()　次の要素を返してカーソルを次に移動

```ruby
import java.util.ArrayList;
import java.util.Iterator;

public class IteratorSample {

	public static void main(String[] args) {
		ArrayList<String> list = new ArrayList<String>();
		list.add("Java入門");
		list.add("勉強中");
		list.add("頑張ろう");

		Iterator<String> it = list.iterator();
		while(it.hasNext()) {
			System.out.println(it.next());
		}
	}

}

```

ArrayListのときと出力は同じになるが、iteratorを使うと要素を削除しながらループしたりするときに便利
いろんなコレクションを共通の書き方で扱える仕組みみたいなので、これから先勉強が進んだ時に便利と思える瞬間がくるはず！！
