## クラス

Javaは複数のクラスを組み合わせてプログラムを構築する。

```ruby
public class Cat {
	String name = "ミー"; // フィールドに定義された変数のことおをメンバ変数という
	
	public String cry() {
		int num = 0; // ローカル変数に定義された変数のことをローカル変数という
		...省略...
	}
}
```

※ローカル変数は定義したメソッド内でしか参照できない

※メンバ変数はクラス内のすべてのメソッドから参照できる

### クラスの使い方

```ruby
クラス名 変数名　＝ new クラス名（引数）;
```

例：まずは、部品のクラスを作る

```ruby
public class Cat {
	String name = "ミー";
	
	public String getCry() {
		return "にゃー";
	}
}

```

そして、そのクラスを呼び出す

```ruby
	public static void main(String[] args) {
		Cat cat = new Cat();
		
		String name = cat.name;
		System.out.println("猫の名前:" + name);
		System.out.println("猫の鳴き声:" + cat.getCry());
		
	}
	
	// 出力
	// 名前:ミー
	// 猫の鳴き声:にゃー

```

クラスを作ったらnew演算子でオブジェクトを生成する！

### メソッド

Javaでは戻り値の型を指定しなければならない。

例：

```ruby
int add(int a, int b) {
    return a + b;
}
```

戻り値がない場合はvoidを指定する
```
void greet() {
    System.out.println("Hello");
}
```
