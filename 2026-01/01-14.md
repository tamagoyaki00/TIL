## Javaの学習記録
Reactを使ってなにかアプリを作ろうと思っていたが・・・、就職だったり今後のこと考えてJava分かったほうが、他の言語も理解しやすそう＋求人が多いということで、Javaの学習に切り替えました。
Rubyと多少の違いはあれど、基本的な演算子とかは一緒なので理解しやすい。
なによりもVScodeになれているので、Esclipseが使いにくく感じる。VSCodeでもJavaの開発はできるみたいだが、郷に入っては郷に従えということで、一番使われているというEsclipseを使おう。

JavaのプログラムはJava仮想マシン（JVM）上で動作する。

JVMがOSの違いを吸収してくれるので、**JAVAはプラットフォームに依存しない。**

RubyやPythonはOSごとに実行環境が違い、動くことが多いが、保証されているわけではない。

■ Javaのバージョンについて

- Javaも **Java 8 / 11 / 17 / 21** みたいにバージョンアップする
- ただし **後方互換性をめちゃくちゃ重視** している
- だから古いコードが新しいJavaでも普通に動くことが多い

Rubyとの違い

- Rubyは破壊的変更が起きやすく、バージョンアップで動かなくなることがある
- Javaは互換性を壊さない文化なので、安心してアップデートできる

→**なのでJavaは金融・業務システムなどに使われている。**

## Esclipseでクラスの作成方法

Javaプロジェクト作成を押下

基本的にsrc配下にファイル（クラス）を作っていく

クラスを作成したいファイルを右クリック→クラスを選択→クラス名を入力

★`public static void main(String[] args)` にチェックを入れると便利（Javaはmainメソッドから実行される）

Javaで標準出力で確認してみる

```ruby
public class Hello {

	public static void main(String[] args) {
		System.out.println("Hello World");
	}

}

```

メニューバーの「実行」タブ → 「実行」 → 「Javaアプリケーション」を選択するとコンソールにHello Worldが出力される

★実行はCtrl + F11 でできる

## 変数

参照：https://qiita.com/morioheisei/items/3d27e82d6cd88068dded

Javaの変数にはプリミティブ型と参照型がある

### プリミティブ型（値そのものをもつ）

基本的な数値や真偽値はプリミティブ型

例：

int

double

boolean

char

long

float

short

byte

```ruby
int a = 10;

a の箱の中には「10」という値そのものが入っている
int a = 10;
int b = a;  // b に 10 がコピーされる

```

Javaでは整数値が細かく分かれている。（メモリ節約のため）

| **型** | **サイズ** | **値の範囲** |
| --- | --- | --- |
| byte | 8ビット | -128〜127 |
| short | 16ビット | -32768～32767 |
| int | 32ビット | -2147483648～2147483647 |
| long | 64ビット | -9223372036854775808～9223372036854775807 |

浮動小数点型も同じく

| **型** | **サイズ** | **値の範囲** |
| --- | --- | --- |
| float | 32ビット | ±1.40239846×10^-45～±3.40282347×10^38 |
| double | 64ビット | ±4.94065645841246544×10^-324～±1.79769313486231570×10^308 |

### 参照型（値がある場所をもつ）

クラスから作られるオブジェクトは全部参照型。

例：

String

ArrayList

Scanner

自分で作ったクラス

```ruby
String s = "Hello World";
s の箱の中には「Hello という文字列がある場所（アドレス）」が入っている

String s1 = "Hello";
String s2 = s1;  // s1 と s2 は同じ場所を指す

```

## 演算子

a++ , ++aの違い

```ruby
x = ++a  // aに1を加算した後にxに値を代入

x = a++ xの値を代入した後に1を加算する

		int a1 = 1;
		int x1 = ++a1;
		
		System.out.println(x1);　// 2
		System.out.println(a1); // 2
		
		int a2 = 1; 
		int x2 = a2++;
		
		System.out.println(x2); // 1
		System.out.println(a2); // 2
```

※＋＋は中身を直接書き換える特殊な演算子
