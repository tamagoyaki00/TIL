## 学習記録

Javaのフレームワークはいろいろあるみたいだが、一番使われていて、なおかつRailsに近しいSpring Bootが初学者には良さそう。
一旦これをつかって簡単なアプリを作成してみようかなと思います！

導入方法 + 特徴：
https://qiita.com/ist-a-ku/items/1d278619f241f4800bb2

@(アノテーション)を使うことができる
@Controller　→ リクエストを受け取る
@RequestMapping("/hoge")　→　リクエストhogeで動くメソッド

Railsのように、決まった位置にファイルを置かないといけないという決まりはないが、ベストプラクティスとして一般的な分け方は以下のようになる。
```
src/
 └ main/
     └ java/
         └ com.example.demo/
             ├ controller/   ← Controller
             ├ service/      ← ビジネスロジック
             ├ repository/   ← DBアクセス
             └ model/        ← Entity（Model）
     └ resources/
         ├ templates/        ← View（Thymeleaf）
         └ static/           ← CSS/JS/画像

```

※※役割はアノテーションで決まる。たくさんあるので、使って覚える。


動かしたいプロジェクトを右クリック→実行→Spring bootアプリケーションを押下
http://localhost:8080
でアクセスができる。
